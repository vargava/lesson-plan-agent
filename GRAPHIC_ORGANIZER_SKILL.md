# SKILL: Graphic Organizer Generator

## Purpose
You are a subagent in a weekly lesson plan workflow. Your job is to read a completed lesson plan (already generated by another subagent) and the original raw lesson data from Google Sheets, then produce a compact 1-page teacher reference document called a Graphic Organizer.

This document is a teacher-facing reference that consolidates the five most critical pieces of a lesson on one page for at-a-glance use during class. It is not a student handout.

---

## Inputs You Will Receive

1. **Raw lesson data** — a pipe-delimited cell grid exported from Google Sheets (the `data_a` or `data_b` value from the fetch step)
2. **Generated lesson plan text** — the full contents of `tmp/lesson_a.txt` or `tmp/lesson_b.txt`, already written by the lesson-generation subagent
3. **Output path** — where to write the organizer (e.g., `tmp/organizer_a.txt`)
4. **Lesson identifier** — tab name (e.g., `L10 326 & 327 (ThF)`)

---

## Output Format

Write a plain-text file to the specified output path using this exact structure. Every section header must appear exactly as shown (all caps, with colon, on its own line). Do not add extra headers, sub-headers, or commentary outside this structure.

```
DO_NOW:
[problem 1 — exact text]
[problem 2 — exact text]
[problem 3 — exact text]

OBJECTIVE:
[SWBAT statement(s) — English only]

VOCABULARY:
1. [term] — [brief student-friendly definition, one sentence]
2. [term] — [brief student-friendly definition, one sentence]
3. [term] — [brief student-friendly definition, one sentence]
4. [term] — [brief student-friendly definition, one sentence]
5. [term] — [brief student-friendly definition, one sentence]

ACTIVITY_INSTRUCTIONS:
[exact Board Instructions text from the Independent/Partner Practice block]

SUMMARY:
[closure discussion questions from Block 5]
```

---

## Extraction Rules — Section by Section

### DO_NOW
- **Source:** the lesson plan file (preferred) or raw sheet data (fallback)
- Extract the three numbered Do Now problems from Block 1 of the lesson plan exactly as written in the [EN] section
- Copy the student-facing problem text verbatim — numbers, expressions, word problem text as written
- Do not include teacher instruction lines (e.g., "Project the following...", "Ask students to...") — only the three problems students work on
- Do not paraphrase or summarize

### OBJECTIVE
- **Source:** Section 2 of the lesson plan (Learning Objectives)
- Extract the [EN] SWBAT statements only — not [ES] or [MX] versions
- Include all objectives if there are multiple, one per line
- Use the exact wording from the lesson plan; do not rephrase

### VOCABULARY
- **Source:** Section 3 of the lesson plan (Vocabulary / Key Terms); cross-reference the raw sheet data (Unit Plan tab) for the term list
- Include up to 5 terms — if the lesson has more than 5, prioritize terms that appear in the exit ticket or are flagged as essential in the Unit Plan tab
- Format each line as: `[number]. [Term] — [definition]`
- Definitions must be student-friendly: one sentence, plain language, no textbook jargon
- Example: `1. Ratio — a comparison of two quantities using division`
- Use English throughout — do not include Spanish equivalents

### ACTIVITY_INSTRUCTIONS
- **Source:** the lesson plan file only (not the raw sheet data)
- Locate Block 4 (Independent / Partner Practice) in the lesson plan
- Find the "Board Instructions" or "BOARD INSTRUCTIONS" subsection — it contains the exact text the teacher writes on the board
- Copy verbatim; if the text is formatted inside a box using +--- or similar box-drawing characters, strip those characters and preserve only the text content
- Do not include surrounding teacher narrative (e.g., TEACHER DOES:, PURPOSE:, transition lines)

### SUMMARY
- **Source:** the lesson plan file (preferred); fall back to raw sheet data if Block 5 closure questions are absent
- From Block 5 of the lesson plan, extract the [EN] closure discussion questions — these appear in the SUMMARY portion of "BLOCK 5 — SUMMARY + EXIT TICKET", before the exit ticket is distributed
- Copy the discussion questions as written — the exact words the teacher will say to students
- Do not include the teacher's wrap-up/bridge statement — questions only
- If the generated lesson plan does not contain a Block 5 SUMMARY section, look in the raw sheet data for any wrap-up or closure questions listed and use those instead
- List each question on its own line

---

## Quality Check

Before writing the output file, verify each item below. If any check fails, emit a WARNING comment at the top of the output file before the first section header (e.g., `# WARNING: Could not extract SUMMARY — Block 5 closure questions not found in lesson plan or raw data`), then write the file with the best available content.

- [ ] All five section headers are present and spelled exactly as shown (DO_NOW, OBJECTIVE, VOCABULARY, ACTIVITY_INSTRUCTIONS, SUMMARY)
- [ ] DO_NOW contains exactly 3 items
- [ ] VOCABULARY has at least 3 and no more than 5 items
- [ ] ACTIVITY_INSTRUCTIONS contains literal board text, not a description of what the teacher should write
- [ ] SUMMARY contains at least 2 discussion questions
- [ ] The entire output uses English only — no [ES] or [MX] content
- [ ] No section contains placeholder text or "N/A"

- Lesson B organizer: `tmp/organizer_b.txt`
- Create `tmp/` if it does not exist

After writing the file, confirm: "Organizer written to tmp/organizer_X.txt"
